GREGORY HOSKING - BRAWL STARS WIKI

UNDER CONSTRUCTION!

I will be building a wiki for the Brawl Stars mobile game.

Home Page

The home page will consist of a general overview of the game as well as a navigation menu linking to the other pages.

Content Page 1 - Brawlers

The first content page will consist of descriptions for each of the Brawlers (playable characters) in the game. Each description will include statistics about the Brawler and their abilities as well as images of the Brawler and their different skins (variations).
Because there are currently 44 Brawlers in the game, I will most likely spread this information across multiple pages rather than put it all on one page.

Content Page 2 - Events

The second content page will consist of descriptions for each of the Events (playable game modes) in the game. Each description will include a basic rundown of the rules and objectives of the Event as well as images of the Event in action.
Because there are currently around 10 or so events in the game, I will most likely spread this information across multiple pages rather than put it all on one page.

Content Page 3 - Items

The third content page will consist of descriptions for each of the various items in the game. Each description will include what the item does and how to obtain it as well as images of each item in the game.
Because there are only a handful of items in the game, I will be able to fit this information on one page.
